--- Phân tích ---

Ban yêu cầu --staged-only, tức là chỉ đóng gói các file đã được git add (staged).
Tuy nhiên, kết quả git diff --cached --name-only trả về RỖNG — không có file nào đang ở trạng thái staged.

Tất cả 15 file đang ở trạng thái "modified but not staged" (chưa git add).

--- Kết luận và phản hồi cho người dùng ---

Không có file nào được staged (git add) — không thể chạy build với --staged-only.

Bạn có 2 lựa chọn:

1. Stage các file bạn muốn đóng gói trước, ví dụ:
   git add app/Controllers/Admin/AjaxController.php
   git add packages/skilldo/cms/src/...
   ... rồi chạy lại: php tool/build-update.php --staged-only

2. Đóng gói TẤT CẢ file đã thay đổi (không dùng --staged-only):
   php tool/build-update.php
